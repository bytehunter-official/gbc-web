# 《Girls Band Cry（少女乐队的呐喊）》主题网页
一个基于纯前端技术栈构建的《Girls Band Cry（少女乐队的呐喊）》主题网页，包含角色展示、音乐播放、相册等核心功能，还原并展示该IP的音乐与角色内容。

![预览图](https://github.com/bytehunter-official/gbc-web/blob/637610f1cb7159020b770ff278bd964e3ee6e42e/screenshot.jpeg)

## 项目介绍
本项目是面向GBC观众老爷的静态网页应用，有角色介绍、原声音乐播放、相册展示等内容，采用经典三件套HTML+JS+CSS开发，无后端，可直接下载所有内容上传到服务器。

### 核心功能
- 🎵 **音乐播放**：内置《Girls Band Cry》CD曲目库，均为320kbps的MP3文件，支持曲目切换、播放/暂停、进度展示等基础播放功能
- 🎭 **角色展示**：展示小孩姐、mmk等核心角色的图文信息
- 📸 **相册模块**：预留相册展示区域，可扩展角色/剧情相关图片集
- 📱 **响应式设计**：适配不同设备尺寸的视觉与交互体验
- 🧭 **导航体系**：分模块管理首页、角色、音乐、相册、关于等页面

## 技术栈
- 前端基础：HTML5 + CSS3 + JavaScript（ES6+）
- 样式方案：原生CSS（模块化拆分：基础样式、滚动样式、页面样式）
- 图标资源：自定义iconfont字体图标
- 媒体处理：HTML5 Audio 音频播放

## 快速开始
### 运行方式
1. 克隆/下载本项目到本地
2. 直接打开 `index2.html` 文件（推荐使用Chrome/Firefox等现代浏览器）
3. 或通过本地静态服务器运行（如VSCode Live Server、http-server等）：
   ```bash
   # 安装http-server（需Node.js环境）
   npm install -g http-server
   # 进入项目根目录启动
   http-server ./ -p 8080
   # 访问 http://localhost:8080/index2.html
   ```

## 扩展与定制
1. **添加新曲目**：
   - 在 `music/` 目录放入新音频文件
   - 在 `index2.html` 的 `scroll_3` 模块中新增 `<li>` 节点，配置封面图、曲目名、乐队名
   - 同步更新 `palyer.js` 中的曲目映射逻辑
2. **新增角色**：
   - 在 `image/` 目录添加角色图片
   - 在 `index2.html` 的 `scroll_2` 模块中新增角色图片节点，配置对应 `data-iid` 等属性
   - 调整 `click-box.js` 中的角色切换逻辑
3. **样式定制**：
   - 修改 `css/` 目录下的样式文件，可自定义颜色、布局、动画等视觉效果
